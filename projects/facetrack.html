FaceTrack - Facial Recognition Attendance System

  • Programming Language: Python 3.7
  • Computer Vision: OpenCV
  • Facial Recognition: Face Recognition Library
  • Data Processing: NumPy
  • GitHub URL: Project Link

FaceTrack is a facial recognition-based attendance system that captures faces in real-time using a webcam, compares them with pre-trained images, and marks attendance automatically.

Key Features

  • Real-time face detection and recognition
  • Automated attendance marking
  • High accuracy facial matching
  • Simple and intuitive interface
  • Export attendance records
